Customer Segmentation & Behavioral Analysis (EDA)
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on customer data to understand behavior patterns and identify meaningful customer segments. The dataset includes demographic details, profession, family size, and spending behavior.

🎯 Objective


Analyze customer distribution across segments (A, B, C, D)


Identify factors influencing spending behavior


Understand key customer patterns and differences


Find useful features for future modeling

🛠️ Tools Used


Python

Pandas

NumPy

Matplotlib

Seaborn


📊 Key Insights


Older customers generally show higher spending scores

Smaller families are more common and show different spending behavior

Work Experience does not strongly impact customer segmentation

Each segment shows distinct patterns in age, family size, and spending


👥 Customer Segments

Segment D: Young customers with low spending behavior

Segment C: Older customers with high spending value

Segment B: Middle group with mixed behavior

Segment A: Stable small households with moderate spending

💡 Business Impact

Helps design targeted marketing strategies

Supports personalized offers for different customer groups

Provides foundation for future machine learning models

📌 Conclusion

This analysis helps understand customer behavior clearly and supports better business decision-making.
